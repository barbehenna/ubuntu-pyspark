# Readme

This docker image is designed to be a platform for the PrairieLearn autograder for UIUC STAT 480. It is based on Ubuntu 18.04 and comes with Apache Spark, Python 3.7, and the following python packages:

- numpy  
- pandas  
- scipy  
- scikit-learn  
- matplotlib  
- seaborn 
- sqlalchemy  
- jupyter
